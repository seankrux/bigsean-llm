# BigSean LLM Gateway

Self-hosted AI assistant platform with multi-provider LLM support, document embedding, RAG, and workspace-based conversations.

## Features

- **Multi-provider LLM support** — Ollama, Gemini, Groq, Mistral, xAI, Moonshot
- **Document embedding & RAG** — Upload documents and query them with retrieval-augmented generation
- **Workspace-based conversations** — Organize chats into isolated workspaces with independent context
- **Agent capabilities** — Autonomous task execution with tool use
- **REST API** — Programmatic access to all platform features
- **Custom gold-themed branding** — Personalized UI with gold accent styling
